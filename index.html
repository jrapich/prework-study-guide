CSS

  • CSS is used to provide styling to a webpage.
  • CSS can be applied to a specific element in HTML, such as paragraphs p, unordered lists ul, etc.
  • think of each element as a "box" and we can define the parameters and colors of that box
  • margin is the space around that box/element
  • padding is extra space around the content inside the box/element
  • we can also assign classes to stuff in HTML and then apply CSS to specific HTML class

GIT

  • Some useful commands related to branches:
    • git status: checks what branch we are currently on
    • git checkout: switchest to a specified branch
      • the -b switch creats a new branch at the same time and switches to it. syntax:
      • git checkout -b new-branch-name
    • git pull updates your current branch from the repo, or the branch you specify
      • what does "origin do exactly? syntax:
      • git pull origin main
    • git add -A commits all local files to the current branch
    • git commit -m "message" commits the current branch to repo, -m switch addes a description
    • at this point rememebr to frequently do git pull origin main so you always have the up to date main branch to avoid collab errors/working on outdated code
    • git push origin branch-name-here pushes the branch to the repo
